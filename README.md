# Калькулятор ЖКХ (показания счётчиков)

Расчёт коммунальных платежей по логике Excel: ХВС, ГВС, водоотведение, подогрев, электричество день/ночь.

## Запуск без сборки

1. Установите Python 3.8+.
2. В папке `counter_calc` выполните:
   ```
   python main.py
   ```

## Сборка в .exe (один файл, любой компьютер)

1. Установите Python и откройте командную строку в папке `counter_calc`.
2. Запустите:
   ```
   build.bat
   ```
   или вручную:
   ```
   pip install pyinstaller
   pyinstaller --onefile --noconsole --name "Kalkulyator_ZHKH" main.py
   ```
3. Готовый файл: `dist\Kalkulyator_ZHKH.exe`.

Скопируйте `Kalkulyator_ZHKH.exe` на любой компьютер с Windows — запускается без установки Python.  
Данные хранятся в `objects.json` рядом с exe. При первом запуске старые `config.json` и `history.json` автоматически мигрируют в первый объект.

## Несколько объектов (квартир)

- **Вкладки** — переключение между объектами. У каждого объекта свои тарифы и история.
- **«+»** — добавить новый объект (квартира, дача и т.д.).
- **ПКМ на вкладке** — переименовать или удалить объект.

## Ввод данных

- **Предыдущие показания** и **текущие показания** для: ХВС (м³), ГВС (м³), электричество день (кВт·ч), электричество ночь (кВт·ч).
- Кнопка **«Рассчитать»** — считает по формулам из Excel.
- **«Тарифы»** — изменить тарифы для текущего объекта (отдельно для каждой квартиры).

## Логика расчёта (как в Excel)

- Водоотведение = (ХВС+ГВС расход) × тариф водоотведения.
- ХВС = расход ХВС × тариф ХВС.
- Подогрев ГВС = расход ГВС × 0,06 Гкал/м³ × тариф за Гкал.
- ГВС = расход ГВС × тариф ГВС.
- Итого вода = водоотведение + ХВС + подогрев + ГВС.
- Свет = (день × тариф день) + (ночь × тариф ночь).
- **ИТОГО = вода + свет.**
